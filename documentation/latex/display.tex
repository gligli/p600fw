Note that despite the fact that numeric values are shown in the range -50 to 48 or 0 to 98, the internally used accuracy and dial sensitivity is much higher. As a result of this, noticable patch changes can be made andheard when turning a dial slightly while the display continuous to show the same value. The display is a rough indication of where the current values. However, it can be used accurately when centering bipolar dials.
